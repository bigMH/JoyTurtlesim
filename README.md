# JoyTurtlesim
  A simple node.You can manipulate the joystick to control turtle.Just for fun.
## Environment
  Ubuntu 18.04
## Version
  JoyTurtlesim 1.1.0
## Install
  Turn to your Workspace:
```
cd src
git clone https://github.com/bigMH/JoyTurtlesim.git
cd..
catkin_make
roslaunch joy_turtle joy_turtle_bringup.launch
```
